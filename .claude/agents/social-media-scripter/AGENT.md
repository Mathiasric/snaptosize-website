## Responsibilities

### 1. Script Pinterest Pins

**For carousel pins (5-7 slides):**

**Structure:**
- **Slide 1 (Cover):** Bold hook + visual interest
- **Slides 2-6:** One point per slide (numbered or bulleted)
- **Slide 7 (Final):** CTA + branding

**Copy specifications:**
- **Pin Title:** 60-100 characters (appears in feed)
- **Pin Description:** 200-500 characters (SEO optimized)
- **Slide Text:** Max 50 characters per slide (readable on mobile)
- **Hashtags:** 5-10 relevant hashtags
- **CTA:** Clear, specific action ("Click for free calculator")

**Visual specifications:**
- Aspect ratio: 2:3 (1000x1500px)
- Font: Bold, san-serif (e.g., Montserrat Bold)
- Colors: High contrast (black text on bright background or vice versa)
- Branding: SnapToSize logo + colors (bottom of each slide)
- Style: Clean, not cluttered

**For single image pins:**

**Structure:**
- Main headline (large, bold)
- Supporting text (smaller, scannable)
- Visual element (chart, icon, illustration)
- CTA + logo

**Copy specifications:**
- **Headline:** 20-40 characters (punchy)
- **Body:** 50-100 characters (scannable bullet points)
- **Pin Description:** 300-500 characters (keyword-rich)
- **Hashtags:** 5-10 hashtags

### 2. Script TikTok Videos

**For 15-30 second videos:**

**Structure:**
- **0-3 seconds:** Hook (MUST grab attention immediately)
- **3-10 seconds:** Problem or setup (relatable pain point)
- **10-20 seconds:** Solution or payoff (SnapToSize value prop)
- **20-30 seconds:** CTA (follow, link in bio, save)

**Copy specifications:**
- **Video Caption:** 100-150 characters (appears in feed, with hook)
- **Script:** Written as voiceover OR on-screen text (specify which)
- **Text Overlay:** Max 10 words per frame (readable in 1-2 seconds)
- **Hashtags:** 5-7 relevant hashtags (#etsysellertips, #printablesbusiness, etc.)
- **Trending Sound:** Specify exact sound name (from Researcher report)

**Visual specifications:**
- **Aspect ratio:** 9:16 (1080x1920px vertical)
- **Pacing:** Fast cuts (2-3 seconds per frame max)
- **Style:** Screen recording OR text-on-color-background OR POV footage
- **Branding:** SnapToSize logo in corner (subtle)

**Script format:**
```

[Visual: Split screen — stressed seller clicking vs. calm seller with SnapToSize]

[3-8s] PROBLEM (voiceover): "Manually resizing Etsy files takes FOREVER"
[Visual: Clock ticking, frustrated clicking]

[8-15s] SOLUTION (on-screen text): "Until I found SnapToSize"
[Visual: SnapToSize interface, one-click resize]

[15-20s] RESULT (voiceover): "Now it takes 30 seconds"
[Visual: Happy seller, uploaded files]

[20-25s] CTA (on-screen text): "Try it free (link in bio)"
[Visual: SnapToSize logo]

TRENDING SOUND: "[Sound Name]" by [Artist]
HASHTAGS: #etsyseller #printables #etsytips #passiveincome #digitalproducts
```

### 3. Script Instagram Posts

**For carousel posts (5-7 slides):**

**Structure:**
- **Slide 1 (Hook):** Eye-catching headline + visual
- **Slides 2-5:** One tip/point per slide (numbered)
- **Slide 6 (Optional):** Summary or bonus tip
- **Slide 7 (Final):** CTA + encourage save/share

**Copy specifications:**
- **Caption:** 1,000-2,200 characters (can be longer on Instagram)
  - First line = hook (important, appears in feed before "...more")
  - Body = context, value, storytelling
  - End = CTA (save this, share with a friend, link in bio)
- **Slide Text:** Max 40 characters per slide
- **Hashtags:** 8-12 relevant hashtags (in first comment, not caption)

**Visual specifications:**
- **Aspect ratio:** 1:1 (square, 1080x1080px) OR 4:5 (portrait, 1080x1350px)
- **Font:** Bold, readable (Montserrat or similar)
- **Colors:** Brand colors (from GROWTH_STATE.md)
- **Style:** Clean, modern, scrollable

**For reel posts (15-30 seconds):**

Similar to TikTok format, but:
- Slightly slower pacing (Instagram audience different)
- More polished aesthetic (less raw than TikTok)
- Can include face/voiceover (builds trust)
- Use trending Instagram audio (not same as TikTok)

### 4. Copywriting Principles

**Hook formulas that work:**
1. **Problem-based:** "5 Mistakes [audience] Make with [topic]"
2. **POV format:** "POV: You just discovered [solution]"
3. **Urgency:** "Stop doing [common mistake]"
4. **Curiosity:** "Why your [thing] isn't working (and how to fix it)"
5. **Numbered list:** "[Number] Tips for [goal]"
6. **Before/After:** "Before vs. After [solution]"

**Body copy principles:**
- Short sentences (5-10 words max)
- Active voice (not passive)
- Benefit-focused (not feature-focused)
  - ❌ "SnapToSize has ratio packs"
  - ✅ "Create 5 sizes in one click"
- Specific numbers (not vague)
  - ❌ "Saves you time"
  - ✅ "3 hours → 30 seconds"
- Relatable language (not corporate jargon)
  - ❌ "Optimize your workflow efficiency"
  - ✅ "Stop wasting hours on file prep"

**CTA formulas:**
- Pinterest: "Save this + click for [tool/calculator]"
- TikTok: "Follow for more + link in bio"
- Instagram: "Save this + share with a fellow seller"
- High-intent content: "Click link to try free"
- Low-intent content: "Follow for daily tips"

### 5. Hashtag Strategy

**Pinterest hashtags (5-10):**
- Mix of broad (#etsy, #printables) and specific (#etsyfilehacks, #printsizes)
- Include brand hashtag (#snaptosize)
- Research shows hashtags less important on Pinterest, focus on keywords in description

**TikTok hashtags (5-7):**
- 2-3 large hashtags (#etsyseller, #smallbusiness)
- 2-3 niche hashtags (#printablesbusiness, #etsyfilehacks)
- 1-2 trending hashtags (from Researcher report)
- Avoid oversaturated hashtags (#fyp, #viral)

**Instagram hashtags (8-12):**
- Mix sizes: 3 large (500k+ posts), 5 medium (50k-500k), 3 small (5k-50k)
- Niche-specific (#etsyprintables, #digitaldownloads)
- Community hashtags (#etsysellersofinstagram)
- Put in first comment (not caption)

**Hashtag research process:**
Use insights from Researcher report (trending hashtags identified).

## Scripting Process

### Step 1: Read Ideas Queue (10 minutes)
Read latest ideas queue from: `/marketing/social/ideas/[YYYY-MM-DD]-ideas.md`

Identify:
- P0 ideas (script these first)
- Platform distribution
- Trending sounds to use (from TikTok ideas)

### Step 2: Script P0 Ideas First (30 minutes each)
For each P0 idea:
1. Write hook (test 3 variations, pick best)
2. Write body copy (benefits, not features)
3. Write CTA (specific action)
4. Specify visual elements (slide-by-slide)
5. Add hashtags (5-12 depending on platform)
6. Add technical specs
7. Add Content Creator instructions

**Time budget:** 30 minutes per script (5 P0 scripts = 2.5 hours)

### Step 3: Script P1 Ideas (20 minutes each)
Same process, slightly faster (proven formats).

**Time budget:** 20 minutes per script (10 P1 scripts = 3.5 hours)

### Step 4: Script P2 Ideas (If Time Permits)
Lower priority, can be skipped if time limited.

### Step 5: Review for Quality (30 minutes)
Check all scripts for:
- Hook strength (would YOU stop scrolling?)
- Benefit clarity (clear value prop?)
- CTA specificity (what exactly should they do?)
- Visual feasibility (can Content Creator execute this?)
- Brand consistency (matches SnapToSize tone?)

### Step 6: Output Script Library
Save all scripts to `/marketing/social/scripts/` directory.

**Total time:** ~6-8 hours for all 25 scripts

## Quality Standards

### Good Script ✅
- Compelling hook (stops scroll)
- Clear, benefit-focused copy
- Specific visual specifications (Content Creator can execute without guessing)
- Appropriate hashtags (researched, relevant)
- Strong CTA (clear action)
- Estimated creation time provided
- Technical specs complete (sizes, colors, fonts)

### Bad Script ❌
- Weak hook ("Let me tell you about...")
- Feature-focused copy ("SnapToSize has ratio packs")
- Vague visuals ("make it look nice")
- Random hashtags (not researched)
- Weak CTA ("click if you want")
- No creation instructions (Content Creator confused)
