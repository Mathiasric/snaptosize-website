## Research Summary

### Search Landscape Findings

1. **"resize images for etsy photoshop"** — Dominated by generic tutorials (Adobe community threads, blog posts teaching manual Photoshop Actions setup). Adobe's own documentation covers batch resize but requires multi-step Action creation. No results directly compare Photoshop to a purpose-built Etsy tool. Key insight: Adobe community forums show frustrated sellers asking how to batch resize for multiple ratios — Photoshop Actions only handle one target size per action.

2. **"photoshop alternative etsy prints batch resize"** — Results surface generic alternatives (GIMP, BatchPhoto, Canva, XnConvert) and two Etsy-specific tools: Artigo (getartigo.com) and Photoshop batch resize scripts sold on Etsy itself ($5-15 one-time). Nobody has a head-to-head comparison page positioning a SaaS tool against Photoshop for this exact use case. Major content gap.

3. **"resize etsy printables without photoshop"** — Little Gold Pixel, ShootingStar SVG, and generic resize tool pages rank. All show manual one-at-a-time workflows. The throughput problem (30+ sizes per artwork) is never addressed. SnapToSize's existing page `/how-to-sell-printables-on-etsy-without-photoshop` ranks here but doesn't provide a direct Photoshop comparison.

4. **"photoshop batch resize for etsy sellers"** — Photoshop scripts sold on Etsy ($5-15) rank prominently, plus tutorial blogs (Envira Gallery, Digital Photography School). Real-world data from these pages: manual resize = ~40 seconds per file, batch Action setup = 30-60 min first time. One seller testimonial: "400 files took a full day" before automation. This validates the pain point.

5. **"photoshop batch actions etsy print sizes multiple ratios"** — Very thin results. This is the killer gap: Photoshop Actions only handle ONE target dimension per action. For 5 different ratios with 28 sizes, you need either 28 separate actions or a complex script. Nobody explains this limitation clearly. SnapToSize solves it in one upload.

5. **Does SnapToSize replace Photoshop?** — No. Photoshop is a comprehensive image editor for creating and editing artwork. SnapToSize handles one specific task: converting finished artwork into all standard Etsy print sizes. If you use Photoshop for design, you can add SnapToSize for production.

6. **What if I already have Photoshop — why would I pay for SnapToSize too?** — Time. Even with Photoshop Actions set up, processing one artwork through 28+ sizes takes 5-20 minutes of manual work.
