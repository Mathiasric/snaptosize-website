## FAQ Questions (from People Also Ask + research)

1. **What are the best print sizes for a gallery wall on Etsy?**
   Answer: The five most popular gallery wall sizes on Etsy are 8x10, 11x14, 16x20, 5x7, and 12x16. Top-selling gallery wall listings include all five sizes plus international options (A4, A3) so buyers can choose the perfect fit for their wall. Including 8-15 sizes per listing maximizes conversion.

2. **How many sizes should I offer in a gallery wall set listing?**
   Answer: The top-performing gallery wall listings on Etsy offer 8-15 sizes per print. At minimum, include 5x7, 8x10, 11x14, 16x20, and A4. More sizes = higher perceived value, premium pricing, and fewer "do you have this in..." messages from buyers.

3. **What is the best size for a set of 3 prints above a sofa?**
   Answer: For a standard sofa (72-84" wide), a triptych of 11x14 or 16x20 prints works best. The total gallery wall width should be about 2/3 of the sofa width. Three 16x20 prints with 2-3" gaps span about 52-54" — ideal for a 78" sofa. Three 11x14 prints with gaps span about 37-39" — better for apartment-sized sofas or loveseats.

4. **Should all prints in a gallery wall set be the same size?**
   Answer: It depends on the arrangement style. Grid layouts and triptychs require all prints at the same size for a clean, symmetrical look. Salon-style and stepped arrangements intentionally mix sizes (e.g., one 16x20 anchor with 11x14 and 8x10 supporting pieces). The best strategy is to offer your gallery wall set in multiple size options so buyers can choose their preferred arrangement.

5. **What sizes do I need for a salon-style gallery wall?**
   Answer: Salon-style gallery walls use 5-12 prints in 3-4 different sizes arranged organically. A popular combination is one 16x20 (anchor), two 11x14 (mid-size), two 8x10 (supporting), and one or two 5x7 (small accents). Start with the largest piece slightly off-center, then build around it. Having every print available in all sizes lets buyers customize their salon wall arrangement.

6. **How much space should I leave between gallery wall prints?**
   Answer: The standard spacing for gallery wall prints is 2-3 inches (5-7.5 cm) between frames. For small frames (5x7, 8x10), use 2" gaps. For larger frames (16x20+), use 2.5-3" gaps. For grid layouts, keep all gaps identical for a clean look. Include this spacing guidance in your Etsy listing description — buyers appreciate it and it reduces support questions.

7. **What file format should gallery wall digital downloads be?**
   Answer: JPEG at 300 DPI is the standard for Etsy gallery wall digital downloads. JPEG files are compatible with every print service (Walgreens, CVS, Costco, Shutterfly, local print shops) and keep file sizes manageable for Etsy's 20MB per file limit. SnapToSize outputs all files as 300 DPI JPEGs organized in ratio-based ZIP packs.

8. **How do I price a gallery wall set with multiple sizes on Etsy?**
   Answer: Gallery wall sets with all sizes included typically sell for $5-15, while single-size sets sell for $3-8. The more sizes you include, the higher the justified price. A set of 6 prints with 10+ sizes each (60+ files total) can command $10-15. This is significantly more than single prints ($2-5), making gallery wall bundles one of the most profitable formats for Etsy digital download sellers.

### Gallery Wall Arrangement Data
- **Triptych (Set of 3, same size):** Most popular arrangement. Best sizes: 8x10, 11x14, 12x16, 16x20. Three 16x20 prints with 2-3" gaps span ~52-54". Three 11x14 prints span ~37-39".
