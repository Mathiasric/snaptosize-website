1. **What pixel dimensions do I need for a 4×6 print at 300 DPI?**
   - Answer: 1200×1800 pixels exactly. Calculation: 4×300=1200, 6×300=1800. Typical JPEG file size: 0.5-1.5 MB.

2. **What aspect ratio is 4×6?**
   - Answer: 2:3 (0.667 decimal). This is the same ratio as 6×9, 8×12, 12×18, 16×24, 20×30, and 24×36. One source file at 2:3 covers all of these sizes.

3. **Is 4×6 too small for Etsy wall art?**
   - Answer: For standalone wall art, yes — most buyers prefer 8×10 or larger for single-piece display. But 4×6 excels as mini art for shelves, desks, and gallery wall clusters. It's also the standard postcard and greeting card size, which is a massive Etsy category.

4. **Should I include 4×6 in my Etsy printable listing?**

5. **What size frame fits a 4×6 print?**
