## 7. FAQ Topics (5 Questions)

1. **What pixel dimensions do I need for a 16×20 print at 300 DPI?**
   - Answer: 4800×6000 pixels exactly. Calculation: 16×300=4800, 20×300=6000. File size: 8-12 MB as JPEG.

2. **What aspect ratio is 16×20?**
   - Answer: 4:5 (0.8 decimal). Same ratio as 8×10, 12×15, 20×25, 24×30. All scale from same source without cropping.

3. **Does 16×20 fit a standard frame?**
   - Answer: Yes — standard size at IKEA, Target, Amazon, Michaels. Can also be matted in 20×24 or 24×30 frames for gallery presentation.

4. **What file format should I use for 16×20 Etsy prints?**
   - Answer: JPEG at 300 DPI with sRGB color profile. PNG creates 15-20 MB files with no print quality benefit — wastes ZIP space and risks hitting 20MB limit.

5. **Should I include bleed for 16×20 prints?**
   - Answer: No for home printing (buyers print exact size). Optional 0.125" bleed for professional print services = 4875×6075 pixels at 300 DPI.
