## Platform-Specific Rules

### Pinterest
- Max image size: 1000×1500 (2:3 ratio)
- NO hashtags (Pinterest removed them)
- USE keyword-rich descriptions (Pinterest = search engine)
- Always include destination URL
- Board names: "Etsy Print Size Guides", "Etsy Seller Tools", "Digital Prints for Etsy Sellers", "Printable Wall Art Business"

### Instagram
- Feed post: 1080×1350 (4:5) or 1080×1080 (1:1)
- Max 5 hashtags
- Include CTA "Link in bio"
- First line = hook (most important, appears in preview)
- Use line breaks for readability

### TikTok / Reels
- 1080×1920 (9:16)
- 15-45 seconds (sweet spot: 30s)
- Hook in first 1.7 seconds
- Same video file for both platforms
- CTA: "Link in bio → snaptosize.com"
- Series format: "Print Tips · Day X"
