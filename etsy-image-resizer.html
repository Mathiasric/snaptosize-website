FAQ

Does SnapToSize crop my artwork?

No. SnapToSize does not crop. Images are resized to exact print dimensions without cutting off edges.

Is this only for Etsy?

It’s optimized for Etsy-style digital print listings, but the exports work anywhere you need standard print sizes.

What file format do I get?

JPEG files, optimized for upload and file size. The exports include 300×300 DPI metadata.

What about Etsy upload limits?

Etsy has file size limits. SnapToSize enforces a 20MB per ZIP cap to prevent upload issues. If a ZIP exceeds the limit, the app stops and warns you.
