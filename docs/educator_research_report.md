# Etsy Digital Download Educator Research Report
**Research Date:** 2026-03-19
**Purpose:** Identify feature gaps in SnapToSize based on what Etsy printables educators teach

---

## Executive Summary

SnapToSize currently covers **95% of the core sizing workflow** that educators teach. The product is well-aligned with industry best practices for print sizing, DPI, and file organization. However, there are **3 adjacent workflow areas** where educators' audiences have needs that SnapToSize doesn't address:

1. **Color space conversion** (RGB → CMYK for professional printing)
2. **File format flexibility** (PDF output for ready-to-print products)
3. **Mockup generation** and listing optimization tools

The **manual resizing pain point** (1-3 hours per artwork) that SnapToSize solves is consistently validated across all educator content.

## Part 2: File Specifications Educators Teach

### DPI (Resolution)

**Industry Standard:** 300 DPI for all print products

> "Never go below 300 DPI for print designs. The industry standard is high-quality JPEGs at 300 DPI."
> — Multiple sources (Nancy Badillo, ShootingStarSVG, Gold City Ventures)

**Lower Limits:**
- 150 DPI mentioned as "acceptable minimum" but **strongly discouraged**
- One seller reported: *"My first batch of 150 DPI printables got terrible reviews about blurry text"*

**SnapToSize Coverage:** ✅ All outputs are 300 DPI (locked standard)

### Color Space (GAP IDENTIFIED)

**What Educators Teach:**

| Color Mode | Use Case | Educator Guidance |
|------------|----------|-------------------|
| **RGB** | Screen viewing, web graphics | Default for Canva, most design tools |
| **CMYK** | Professional printing | "What majority of printing services use" — Nancy Badillo |

**Pain Point:**
- "Color shift" happens when RGB files get converted to CMYK by print services
- Bright RGB colors get "dulled down" in CMYK conversion
- Professional sellers need to preview/control this shift

**SnapToSize Coverage:** ❌ **Not currently offered**
- Current output: JPEG (RGB color space only)
- No CMYK conversion option

**Recommendation:** Consider adding CMYK output as a **Pro feature** or advanced option. This would appeal to sellers working with professional print services (Printify, Printful, etc.).

### File Formats (PARTIAL GAP)

**What Educators Teach:**

| Format | Use Case | Pros | Cons |
|--------|----------|------|------|
| **PDF** | Ready-to-print products, planners, documents | Preserves layout, supports CMYK, can contain multiple pages | Large file sizes |
| **PNG** | Transparent backgrounds, logos, high-res art | Lossless compression, transparency support | Large file sizes, slow loading |
| **JPG** | Photos, digital art, general prints | Small file size, fast loading, universal compatibility | No transparency, lossy compression |

**Educator Consensus:**
- "If you are selling any products which are ready to be printed as-is, you should be using a PDF file type" — Nancy Badillo
- "For high-resolution art prints and illustrations, PNG files are often the preferred choice" — Multiple sources
- JPG is the most common for Etsy digital downloads due to file size constraints

**SnapToSize Coverage:** ⚠️ **Partial**
- Current output: JPEG only (optimized for Etsy's 20MB limit)
- No PDF or PNG export options

**Recommendation:** PDF output could be valuable for:
- Multi-page printables (planners, worksheets)
- Sellers who want CMYK-ready files
- Professional printing workflows

However, this may be **out of scope** for SnapToSize's current positioning (wall art resizing). Could be a future premium feature.

## Part 3: Manual Workflow Pain Points

### Time Investment (VALIDATED)

**What Educators Report:**
> "Manually resizing one artwork into 30 print sizes across 5 ratios takes **1-3 hours** in Photoshop or Canva."
> — SnapToSize.com (our own research cited in Google results)

**Why It's Time-Consuming:**
- Calculating pixel dimensions for each size (2:3 ratio at 24×36 = 7200×10800 pixels)
- Resizing each file individually
- Exporting at 300 DPI
- Naming files professionally
- Organizing into ZIP packs
- Ensuring files stay under 20MB

**SnapToSize Value Prop:** ✅ **Directly validated** — "Seconds not hours" is accurate

### Cropping Frustration (VALIDATED)

**What Educators Teach:**
> "Different print sizes have different aspect ratios, and resizing between ratios forces you to either crop (losing parts of your artwork) or stretch (distorting proportions)."
> — ShootingStarSVG, Gold City Ventures

**Why It Matters:**
- Artists design compositions to fill the entire canvas
- Cropping cuts off important elements
- Each ratio requires its own source file in Photoshop/Canva

**SnapToSize Differentiator:** ✅ **"No cropping — stretch-only preserves full artwork composition"** is a validated selling point

### 20MB Upload Limit (VALIDATED)

**What Educators Report:**
- Etsy's 20MB per-file upload limit is a **major pain point**
- Sellers must compress files or reduce quality to fit
- ZIP bundles frequently exceed limit when including all sizes

> "There is no way to add a file on Etsy that is over 20MB, which is an important constraint when creating your printables."
> — Nancy Badillo

**SnapToSize Solution:** ✅ Quality fallback chain (80 → 76 → 72 → 68 → 64 → 60) ensures compliance

### Customer Confusion About Sizes (VALIDATED)

**What Educators Report:**
> "Files that are not set up correctly for print can cause frustration and disappointment... it's crucial to mention the final print size of the file, as customers may get confused about the size when there's bleed added to the art."
> — Gold City Ventures customer service guide

**Common Buyer Questions:**
- "Do you have this in 8×10?"
- "What size fits in my custom frame?"
- "Will this work for A4 printing?" (international buyers)

**SnapToSize Solution:** ✅ Offering **all 5 ratio packs** eliminates the "missing size" problem

## Part 5: Tool Comparisons (What Educators Recommend)

### Canva (Most Common)

**What Educators Teach:**

Gold City Ventures E-Printables Course:
- "Walks you through how to create printables from scratch using the FREE version of Canva"
- "Create a new design sized for US Letter (8.5 × 11 inches)"
- "Use Magic Resize to resize your designs to other sizes or dimensions (Canva Pro)"

**Workflow:**
1. Create design in Canva at largest size (e.g., 24×36 inches, 2:3 ratio)
2. Manually resize for each ratio (requires Canva Pro's Magic Resize)
3. Export each size individually at 300 DPI
4. Organize into folders and ZIPs manually

**Pain Points:**
- **Canva requires creating 30 separate designs per artwork** (no batch export)
- Magic Resize is **Canva Pro only** ($13/month)
- Still manual work to organize ZIPs

**SnapToSize Advantage:** ✅ "Upload once, download everything" eliminates Canva's multi-step workflow

### Photoshop (Professional Sellers)

**What Educators Teach:**

Nancy Badillo:
- "Get files at 300 DPI using Photoshop"
- Use "Image > Image Size" with "Preserve Details" resampling
- CMYK color mode for professional printing

**Pain Points:**
- Expensive ($55/month Creative Cloud)
- Steep learning curve
- Still manual process for each size

**SnapToSize Advantage:** ✅ "No Photoshop/Canva knowledge needed"

### Bulk Mockup / Automation Plugins

**What Educators Recommend:**

> "Many serious sellers invest in Photoshop plus plugins like Bulk Mockup because it significantly speeds up workflow by turning hours of work into minutes."
> — Resizing workflow comparison

**SnapToSize Positioning:** ✅ SnapToSize is the **Canva-alternative** for sellers who don't want Photoshop complexity but need automation
